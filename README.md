# Project-06

# Features added
+ 3 Mixamo model (Joe, Maynard, and Swat Guy)
+ Zombie AI that chases the player and attacks them
+ Ground has a dirt like texture
+ Swat Guy who you can interact with to learn the story/object of the game
+ Gun selection area with 2 gun options
+ Hold right mouse button for aim down sight
+ Walk and aim animation
+ Idle aim Animation
+ Can only aim when you have a weapon
+ Player grunts when hit
+ Player can survive 1 hit from the zombies and game ends when all enemies are killed or player dies.
+ Start and End screen
+ End Screen shares score
